## [0.0.1] - 2026-01-15

### Added

- Initial project setup with Astro 5.x
- TypeScript configuration
- Tailwind CSS integration
- Biome for linting and formatting
- Basic project structure (components, layouts, pages, assets)
- Welcome component and index page
- Main layout with Astro SVG assets

### Project Structure

```
src/
├── assets/          # Static assets
├── components/      # Reusable components
├── layouts/         # Page layouts
├── pages/           # Route definitions
```

### Tech Stack

- Astro 5.16.9
- TypeScript
- Tailwind CSS
- Biome 2.3.7
